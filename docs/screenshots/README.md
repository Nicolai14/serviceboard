# Screenshots

Diese Bilder werden in der Haupt-[`README.md`](../../README.md) eingebunden. Lege hier einfach PNGs mit **genau diesen Dateinamen** ab, dann erscheinen sie automatisch:

| Datei | Inhalt |
|---|---|
| `dashboard.png` | Dashboard / Server-Übersicht |
| `server-detail.png` | Server-Detailseite (Metriken, 24h-Verlauf) |
| `workflows.png` | Projekt-Workflows (Baukasten-Canvas) |
| `costs.png` | Kostenübersicht |

**Tipp:** Empfohlene Breite ~1400 px, im hellen oder dunklen Theme konsistent. Screenshots können direkt aus dem Browser oder vom Handy stammen — danach `git add docs/screenshots/*.png && git commit && git push`.
